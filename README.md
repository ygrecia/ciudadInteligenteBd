# Ciudad Inteligente – Base de Datos

Proyecto académico desarrollado en la carrera de Informática

## Descripción
Diseño e implementación de una base de datos para una Ciudad Inteligente,
orientada a la gestión de ciudadanos, servicios públicos, movilidad urbana,
seguridad ciudadana y medio ambiente.

## Tecnologías
- MySQL
- MySQL Workbench
- SQL
